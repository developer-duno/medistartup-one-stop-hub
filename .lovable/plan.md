
# 전문가 비교보기 모바일 가독성 개선

## 현재 문제점
현재 비교보기는 가로 테이블(table) 형태로 되어 있어 모바일에서 가로 스크롤이 필요하고, 셀이 좁아 텍스트가 잘리거나 읽기 어렵습니다.

## 개선 방안
모바일(768px 미만)에서는 테이블 대신 **카드 기반 레이아웃**으로 전환합니다. 각 전문가를 세로 카드로 나열하고, 항목별 정보를 카드 내에 라벨-값 쌍으로 표시합니다. 데스크톱에서는 기존 테이블 형태를 유지합니다.

## 변경 내용

### 1. ExpertComparisonView.tsx 수정

- `useIsMobile` 훅을 사용하여 모바일 여부 판별
- **데스크톱**: 기존 가로 테이블 유지
- **모바일**: 전문가별 세로 카드 레이아웃으로 변경
  - 각 카드에 프로필 이미지, 이름, 전문 분야, 서비스(배지), 경력, 프로젝트 수, 지역, 소개, 상세보기 링크 표시
  - 카드 간 구분선으로 시각적 분리
- 하단 버튼도 모바일에서 세로 스택(`flex-col`)으로 배치

## 기술 세부사항

```text
데스크톱 (768px 이상)          모바일 (768px 미만)
+---------------------------+   +------------------+
| 항목  | 김태호 | 이준호  |   | [김태호 카드]    |
|-------|--------|---------|   | 이미지 + 이름    |
| 분야  | ...    | ...     |   | 전문분야: ...    |
| 서비스| ...    | ...     |   | 서비스: 배지들   |
| ...   | ...    | ...     |   | 경력: ...        |
+---------------------------+   | ...              |
                                +------------------+
                                | [이준호 카드]    |
                                | ...              |
                                +------------------+
```
